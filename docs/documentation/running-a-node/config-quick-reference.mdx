# Configuration Quick Reference

## Essential Parameters

```toml
# app.toml
[evm]
evm-chain-id = 262144           # Must differ from Cosmos chain ID
max-tx-gas-wanted = 0           # 0 = unlimited
tracer = ""                     # Empty = disabled

[json-rpc]
enable = true
address = "127.0.0.1:8545"      # HTTP endpoint
ws-address = "127.0.0.1:8546"   # WebSocket endpoint
api = "eth,net,web3"            # Enabled namespaces
allow-unprotected-txs = false   # Require EIP-155 signatures
allow-insecure-unlock = false   # Disable account unlocking
gas-cap = 25000000              # Max gas for eth_call
logs-cap = 10000                # Max logs per query
filter-cap = 200                # Max concurrent filters
```

## Parameter Ranges

| Parameter | Type | Min | Max |
|-----------|------|-----|-----|
| `gas-cap` | uint64 | 0 (unlimited) | uint64 max |
| `filter-cap` | int32 | 0 | int32 max |
| `logs-cap` | int32 | 0 | int32 max |
| `block-range-cap` | int32 | 0 | int32 max |
| `batch-request-limit` | int | 0 | int max |
| `max-open-connections` | int | 0 (unlimited) | int max |
| `evm-timeout` | duration | - | - |
| `http-timeout` | duration | - | - |

## Environment Variables

```bash
export EVMD_JSONRPC_ENABLE=true
export EVMD_JSONRPC_ADDRESS="0.0.0.0:8545"
export EVMD_EVM_EVMCHAINID=1337
export EVMD_JSONRPC_GASCAP=50000000
```

## Health Checks

<CodeGroup>
```bash JSON-RPC
# Check Ethereum JSON-RPC
curl -X POST http://localhost:8545 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"net_version","params":[],"id":1}'
```

```bash WebSocket
# Test WebSocket connection
wscat -c ws://localhost:8546
> {"jsonrpc":"2.0","method":"net_version","params":[],"id":1}
```

```bash Metrics
# Check Prometheus metrics
curl http://localhost:6065/debug/metrics/prometheus
```

```bash Node Status
# Check Tendermint status
curl http://localhost:26657/status | jq .
```
</CodeGroup>
