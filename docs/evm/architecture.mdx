## Overview

The Cosmos EVM module operates in addition to all existing chain features in a self-contained environment, with the ability to modify the state. ######################## REFERENCE ARCHITECTURE DOC WITH ADDITIONAL DETAIL HERE ###########################

It functions as both a gateway to the underlying chain through an EVM client, and a fully-featured EVM layer with Solidity smart contract support and standard EVM JSON RPC functions,while maintaining the security and consensus properties of the base Cosmos chain.


## Core Modules

### x/vm Module

The `x/vm` module is the core EVM implementation module (renamed from `x/evm` in the original evmOS) that provides Ethereum compatibility.

**Capabilities:**
- Provides Ethereum Virtual Machine compatibility
- Handles contract deployment and execution
- Manages EVM state transitions
- Supports Ethereum opcodes

### x/feemarket Module

The `x/feemarket` module implements EIP-1559 fee market mechanism for transaction fee handling.

**Parameters:**
- Base fee change denominator
- Elasticity multiplier
- Min gas price
- Min gas multiplier

**Features:**
- EIP-1559 dynamic fee market mechanism
- Transaction fee handling
