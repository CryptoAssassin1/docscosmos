## Architecture-Based Development

### Core Modules

Based on available documentation, development involves three main modules:

<CardGroup cols={3}>
  <Card title="x/vm Module" icon="microchip">
    Core EVM implementation providing Ethereum compatibility
  </Card>
  <Card title="x/feemarket Module" icon="chart-line">
    EIP-1559 fee market mechanism for transaction fee handling
  </Card>
  <Card title="x/erc20 Module" icon="arrows-rotate">
    Interoperability between Cosmos SDK coins and ERC-20 tokens
  </Card>
</CardGroup>

### Parameter Management

Both VM and FeeMarket modules maintain parameters that can be updated through governance:

**VM Module Parameters:**
- EVM denomination
- Extra EIPs enabled
- Chain configuration (fork schedule)
- Active precompiled contracts

**FeeMarket Module Parameters:**
- Base fee change denominator
- Elasticity multiplier
- Min gas price
- Min gas multiplier

## Smart Contract Development

### Standard Ethereum Development

Since Cosmos EVM is forward-compatible with Ethereum, standard tooling should work:

```bash
# Standard Ethereum development setup
npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox
npm install --save ethers
npm install --save @cosmjs/stargate @cosmjs/proto-signing
```

### Configuration Template

```typescript
// hardhat.config.ts
import { HardhatUserConfig } from "hardhat/config";
import "@nomicfoundation/hardhat-toolbox";

const config: HardhatUserConfig = {
  solidity: {
    version: "0.8.19",
    settings: {
      optimizer: {
        enabled: true,
        runs: 200,
      },
    },
  },
  networks: {
    local: {
      url: "http://localhost:8545", // Standard EVM JSON-RPC port
      accounts: process.env.PRIVATE_KEY ? [process.env.PRIVATE_KEY] : [],
      // ############################
      // # VERIFY: Chain ID and other parameters
      // ############################
    },
  },
};

export default config;
```

## Testing Approaches

### Smart Contract Testing

```javascript
// Standard Hardhat testing pattern
const { expect } = require("chai");
const { ethers } = require("hardhat");

describe("CosmosEVMContract", function () {
  let contract;
  let owner;

  beforeEach(async function () {
    [owner] = await ethers.getSigners();
    
    const Contract = await ethers.getContractFactory("CosmosEVMContract");
    contract = await Contract.deploy("Hello World");
    await contract.deployed();
  });

  it("Should deploy with correct initial state", async function () {
    expect(await contract.message()).to.equal("Hello World");
  });
});
```
