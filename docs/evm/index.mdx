# Introduction to Cosmos EVM

Discover Cosmos EVM, a high-performance execution environment that brings the familiar Ethereum development experience to the modular and interconnected Cosmos ecosystem.

## Critical Differences from Standard EVMs

**Essential for all developers, operators, and teams:** These fundamental differences from Ethereum mainnet and L2s affect smart contract behavior, chain interaction, and infrastructure operation.

### Smart Contract & Transaction Behavior

<CardGroup cols={2}>
  <Card title="Transaction Ordering" icon="clock">
    **Cosmos EVM:** No app-side mempool - transactions ordered by arrival time to proposer. Only one transaction per block per EOA (externally owned account).

    **Standard EVM:** Multiple transactions per block per account, ordered by gas price and nonce in mempool.
  </Card>
  <Card title="Consensus & Finality" icon="shield">
    **Cosmos EVM:** Instant finality via Tendermint BFT consensus (1-2 second blocks). No chain reorganizations.

    **Standard EVM:** Probabilistic finality requiring 12+ confirmations (~12-15 minutes). Risk of chain reorgs.
  </Card>
  <Card title="Precompiled Contracts" icon="zap">
    **Cosmos EVM:** Access native Cosmos SDK modules (staking, IBC, governance) directly from Solidity at predefined addresses.

    **Standard EVM:** Limited to basic cryptographic precompiles (ecrecover, sha256, etc.).
  </Card>
  <Card title="Address Formats" icon="user">
    **Cosmos EVM:** Dual format support - Ethereum (0x123...) and Cosmos (cosmos1...) with deterministic conversion.

    **Standard EVM:** Only Ethereum format addresses supported.
  </Card>
</CardGroup>

### Gas Economics & Fee Structure

| Aspect | Cosmos EVM | Ethereum Mainnet | Ethereum L2s |
|--------|------------|------------------|--------------|
| **Fee Model** | EIP-1559 (base + priority) | EIP-1559 (base + priority) | EIP-1559 or fixed |
| **Gas Token** | Native chain token | ETH | ETH/derivatives |
| **Gas Limit** | Configurable (often higher) | ~30M gas per block | Variable |
| **Precompile Gas** | Much lower costs | Fixed higher costs | Fixed higher costs |
| **Block Time** | 1-2 seconds | 12-15 seconds | Variable |

### RPC & Infrastructure Differences

<CardGroup cols={2}>
  <Card title="JSON-RPC Limitations" icon="code">
    **Supported:** Ethereum RPC methods up to v1.10 including eth_getBalance, eth_call, eth_sendTransaction.

    **Limited:** Advanced debugging methods, mining-specific calls, post-v1.10 methods may not be available.
  </Card>
  <Card title="Mempool Behavior" icon="layers">
    **Cosmos EVM:** Tendermint mempool with first-come-first-served ordering. No traditional MEV opportunities.

    **Standard EVM:** App-side mempool with gas-price ordering. MEV extraction possible.
  </Card>
</CardGroup>

## Quick Start

Get up and running in minutes:

### Connect to Testnet

<Info>
**Cosmos EVM Testnet Configuration**

To connect to the Cosmos EVM testnet, add this network to your wallet:

- **Network Name:** Cosmos EVM Testnet
- **RPC URL:** `https://testnet-evm.cosmos.network`
- **Chain ID:** `9000`
- **Currency Symbol:** `TEST`
- **Block Explorer:** `https://explorer.cosmos.network/evm`
</Info>

<Steps>
  <Step title="Install MetaMask">
    Download and install [MetaMask](https://metamask.io/download/) if you haven't already.
  </Step>

  <Step title="Add Network">
    Open MetaMask and click "Add Network" → "Add a network manually" and enter the configuration above.
  </Step>

  <Step title="Get Test Tokens">
    Visit the [testnet faucet](https://faucet.cosmos.network) to receive test tokens for development.
  </Step>
</Steps>
