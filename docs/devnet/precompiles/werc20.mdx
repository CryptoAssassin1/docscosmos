# WERC20 Precompile

**Address**: Dynamic (assigned per wrapped token)

**Related Module**: Token wrapping utilities

The WERC20 precompile provides wrapped token functionality for native EVM tokens, enabling them to be represented as ERC20-compatible contracts while maintaining a 1:1 backing with the underlying native token.

## Functions

### deposit

**Signature**: `deposit() payable`

**Description**: Deposits native tokens in exchange for wrapped ERC20 tokens.

**Parameters**: None (amount determined by msg.value)

**Returns**: None

### withdraw

**Signature**: `withdraw(uint256 wad)`

**Description**: Withdraws native tokens from wrapped ERC20 tokens.

**Parameters**:
- `wad` (uint256): The amount of native tokens to be withdrawn

**Returns**: None

### totalSupply

**Signature**: `totalSupply() → uint256`

**Description**: Returns the total supply of wrapped tokens.

**Returns**: Total supply amount

### balanceOf

**Signature**: `balanceOf(address account) → uint256`

**Description**: Returns the wrapped token balance for a specific account.

**Parameters**:
- `account` (address): Account to query balance for

**Returns**: Account balance

### transfer

**Signature**: `transfer(address to, uint256 amount) → bool`

**Description**: Transfers tokens to specified address.

**Parameters**:
- `to` (address): Recipient address
- `amount` (uint256): Amount to transfer

**Returns**: Boolean indicating success

### approve

**Signature**: `approve(address spender, uint256 amount) → bool`

**Description**: Approves spender to transfer tokens on behalf of caller.

**Parameters**:
- `spender` (address): Address to approve for spending
- `amount` (uint256): Amount to approve

**Returns**: Boolean indicating success

### transferFrom

**Signature**: `transferFrom(address from, address to, uint256 amount) → bool`

**Description**: Transfers tokens from one address to another using allowance.

**Parameters**:
- `from` (address): Source address
- `to` (address): Destination address  
- `amount` (uint256): Amount to transfer

**Returns**: Boolean indicating success

### allowance

**Signature**: `allowance(address owner, address spender) → uint256`

**Description**: Returns the remaining tokens that spender is allowed to spend on behalf of owner.

**Parameters**:
- `owner` (address): Token owner address
- `spender` (address): Approved spender address

**Returns**: Remaining allowance amount

### name

**Signature**: `name() → string`

**Description**: Returns the name of the token.

**Returns**: Token name

### symbol

**Signature**: `symbol() → string`

**Description**: Returns the symbol of the token.

**Returns**: Token symbol

### decimals

**Signature**: `decimals() → uint8`

**Description**: Returns the number of decimals the token uses.

**Returns**: Number of decimals

### increaseAllowance

**Signature**: `increaseAllowance(address spender, uint256 addedValue) → bool`

**Description**: Increases the allowance granted to spender.

**Parameters**:
- `spender` (address): Address to increase allowance for
- `addedValue` (uint256): Amount to add to current allowance

**Returns**: Boolean indicating success

### decreaseAllowance

**Signature**: `decreaseAllowance(address spender, uint256 subtractedValue) → bool`

**Description**: Decreases the allowance granted to spender.

**Parameters**:
- `spender` (address): Address to decrease allowance for
- `subtractedValue` (uint256): Amount to subtract from current allowance

**Returns**: Boolean indicating success

## Events

- `Deposit(address indexed dst, uint256 wad)`
- `Withdrawal(address indexed src, uint256 wad)`
- `Transfer(address indexed from, address indexed to, uint256 value)`
- `Approval(address indexed owner, address indexed spender, uint256 value)`

<summary>Solidity Interface</summary>

```solidity
// SPDX-License-Identifier: LGPL-3.0-only
pragma solidity >=0.8.18;

import "./../erc20/IERC20MetadataAllowance.sol";

/**
 * @author Evmos Team
 * @title Wrapped ERC20 Interface
 * @dev Interface for representing the native EVM token as a wrapped ERC20 standard.
 */
interface IWERC20 is IERC20MetadataAllowance {
    /// @dev Emitted when the native tokens are deposited in exchange for the wrapped ERC20.
    /// @param dst The account for which the deposit is made.
    /// @param wad The amount of native tokens deposited.
    event Deposit(address indexed dst, uint256 wad);

    /// @dev Emitted when the native token is withdrawn.
    /// @param src The account for which the withdrawal is made.
    /// @param wad The amount of native tokens withdrawn.
    event Withdrawal(address indexed src, uint256 wad);

    /// @dev Default fallback payable function. Must call the deposit method in implementing contracts.
    fallback() external payable;

    /// @dev Default receive payable function. Must call the deposit method in implementing contracts.
    receive() external payable;

    /// @dev Deposits native tokens in exchange for wrapped ERC20 token.
    /// @dev Emits a Deposit Event.
    function deposit() external payable;

    /// @dev Withdraws native tokens from wrapped ERC20 token.
    /// @dev Emits a Withdrawal Event.
    /// @param wad The amount of native tokens to be withdrawn.
    function withdraw(uint256 wad) external;
}
```
