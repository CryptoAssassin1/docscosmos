# Governance Precompile

**Address**: `0x0000000000000000000000000000000000000805`

**Related Module**: [x/gov](https://docs.cosmos.network/main/modules/gov)

The Governance precompile provides comprehensive access to Cosmos SDK governance functionality, enabling smart contracts to participate in on-chain governance through proposal submission, voting, and governance state queries.

## Functions

### submitProposal

**Signature**: `submitProposal(address, bytes, Coin[]) → uint64`

**Description**: Submits a new governance proposal to the network with the specified parameters and optional initial deposit.

**Parameters**:
- `proposer` (address): Address submitting the proposal (must be msg.sender)
- `jsonProposal` (bytes): JSON-encoded proposal data defining the actions to execute
- `deposit` (Coin[]): Array of coins to deposit with the proposal

**Returns**: Unique proposal ID assigned to the new proposal

### vote

**Signature**: `vote(address, uint64, VoteOption, string) → bool`

**Description**: Casts a single vote on an active proposal with the voter's entire voting power.

**Parameters**:
- `voter` (address): Address casting the vote (must be msg.sender)
- `proposalId` (uint64): ID of the proposal to vote on
- `option` (VoteOption): Vote choice (Yes, No, Abstain, or NoWithVeto)
- `metadata` (string): Optional metadata explaining the vote rationale

**Returns**: Boolean indicating successful vote submission

### voteWeighted

**Signature**: `voteWeighted(address, uint64, WeightedVoteOption[], string) → bool`

**Description**: Casts a weighted vote that splits voting power across multiple options with specified weights.

**Parameters**:
- `voter` (address): Address casting the vote (must be msg.sender)
- `proposalId` (uint64): ID of the proposal to vote on
- `options` (WeightedVoteOption[]): Array of vote options with their respective weights
- `metadata` (string): Optional metadata explaining the weighted vote strategy

**Returns**: Boolean indicating successful vote submission

### deposit

**Signature**: `deposit(address, uint64, Coin[]) → bool`

**Description**: Adds additional funds to a proposal's deposit during the deposit period.

**Parameters**:
- `depositor` (address): Address making the deposit (must be msg.sender)
- `proposalId` (uint64): ID of the proposal to deposit to
- `amount` (Coin[]): Array of coins to deposit

**Returns**: Boolean indicating successful deposit

### cancelProposal

**Signature**: `cancelProposal(address, uint64) → bool`

**Description**: Cancels a proposal that is still in the deposit period.

**Parameters**:
- `proposer` (address): Address that submitted the proposal (must be msg.sender)
- `proposalId` (uint64): ID of the proposal to cancel

**Returns**: Boolean indicating successful cancellation

### getProposal

**Signature**: `getProposal(uint64) → ProposalData`

**Description**: Retrieves comprehensive information about a specific proposal including status, voting results, and metadata.

**Parameters**:
- `proposalId` (uint64): ID of the proposal to query

**Returns**: Complete ProposalData struct with all proposal information

### getProposals

**Signature**: `getProposals(uint32, address, address, PageRequest) → (ProposalData[], PageResponse)`

**Description**: Retrieves a filtered and paginated list of proposals based on status and participant filters.

**Parameters**:
- `proposalStatus` (uint32): Filter by proposal status (or 0 for all)
- `voter` (address): Filter proposals where this address has voted (or zero address for all)
- `depositor` (address): Filter proposals where this address has deposited (or zero address for all)
- `pagination` (PageRequest): Pagination parameters for result limiting

**Returns**: Array of matching proposals and pagination information

### getTallyResult

**Signature**: `getTallyResult(uint64) → TallyResultData`

**Description**: Retrieves the current or final vote tally for a proposal showing vote distribution across all options.

**Parameters**:
- `proposalId` (uint64): ID of the proposal to get tally for

**Returns**: TallyResultData struct with vote counts for each option

### getVote

**Signature**: `getVote(uint64, address) → WeightedVote`

**Description**: Retrieves the vote cast by a specific address on a proposal.

**Parameters**:
- `proposalId` (uint64): ID of the proposal
- `voter` (address): Address of the voter to query

**Returns**: WeightedVote struct with the voter's choices and metadata

### getVotes

**Signature**: `getVotes(uint64, PageRequest) → (WeightedVote[], PageResponse)`

**Description**: Retrieves all votes cast on a specific proposal with pagination support.

**Parameters**:
- `proposalId` (uint64): ID of the proposal
- `pagination` (PageRequest): Pagination parameters

**Returns**: Array of WeightedVote structs and pagination information

### getDeposit

**Signature**: `getDeposit(uint64, address) → DepositData`

**Description**: Retrieves deposit information for a specific depositor on a proposal.

**Parameters**:
- `proposalId` (uint64): ID of the proposal
- `depositor` (address): Address of the depositor

**Returns**: DepositData struct with deposit information

### getDeposits

**Signature**: `getDeposits(uint64, PageRequest) → (DepositData[], PageResponse)`

**Description**: Retrieves all deposits made on a specific proposal with pagination support.

**Parameters**:
- `proposalId` (uint64): ID of the proposal
- `pagination` (PageRequest): Pagination parameters

**Returns**: Array of DepositData structs and pagination information

### getParams

**Signature**: `getParams() → Params`

**Description**: Retrieves current governance parameters including deposit requirements, voting periods, and threshold values.

**Returns**: Params struct containing governance configuration

### getConstitution

**Signature**: `getConstitution() → string`

**Description**: Retrieves the current governance constitution document.

**Returns**: String containing the constitution text

## Events

- `SubmitProposal(address indexed proposer, uint64 proposalId)`
- `CancelProposal(address indexed proposer, uint64 proposalId)`
- `Deposit(address indexed depositor, uint64 proposalId, Coin[] amount)`
- `Vote(address indexed voter, uint64 proposalId, uint8 option)`
- `VoteWeighted(address indexed voter, uint64 proposalId, WeightedVoteOption[] options)`

## Data Structures

### VoteOption
Enumeration of valid vote options:
- `Unspecified` (0): No-op vote option
- `Yes` (1): Support the proposal
- `Abstain` (2): Abstain from voting but participate in quorum
- `No` (3): Oppose the proposal
- `NoWithWeto` (4): Strong opposition with veto power

### WeightedVoteOption
- `option` (VoteOption): Vote choice
- `weight` (string): Vote weight allocation

### WeightedVote
- `proposalId` (uint64): Proposal ID
- `voter` (address): Voter address
- `options` (WeightedVoteOption[]): Vote options with weights
- `metadata` (string): Vote metadata

### DepositData
- `proposalId` (uint64): Proposal ID
- `depositor` (address): Depositor address
- `amount` (Coin[]): Deposit amount

### TallyResultData
- `yes` (string): Yes votes count
- `abstain` (string): Abstain votes count
- `no` (string): No votes count
- `noWithVeto` (string): No with veto votes count

### ProposalData
- `id` (uint64): Proposal ID
- `messages` (string[]): Proposal messages
- `status` (uint32): Current proposal status
- `finalTallyResult` (TallyResultData): Final vote tally
- `submitTime` (uint64): Submission timestamp
- `depositEndTime` (uint64): Deposit period end time
- `totalDeposit` (Coin[]): Total deposited amount
- `votingStartTime` (uint64): Voting period start time
- `votingEndTime` (uint64): Voting period end time
- `metadata` (string): Proposal metadata
- `title` (string): Proposal title
- `summary` (string): Proposal summary
- `proposer` (address): Proposer address

### Params
- `votingPeriod` (int64): Duration of voting period
- `minDeposit` (Coin[]): Minimum deposit required
- `maxDepositPeriod` (int64): Maximum deposit period duration
- `quorum` (string): Minimum participation rate required
- `threshold` (string): Minimum yes vote percentage for passage
- `vetoThreshold` (string): Minimum veto percentage to fail proposal
- `minInitialDepositRatio` (string): Minimum initial deposit ratio
- `proposalCancelRatio` (string): Proposal cancel ratio
- `proposalCancelDest` (string): Proposal cancel destination
- `expeditedVotingPeriod` (int64): Expedited voting period duration
- `expeditedThreshold` (string): Expedited proposal threshold
- `expeditedMinDeposit` (Coin[]): Minimum deposit for expedited proposals
- `burnVoteQuorum` (bool): Whether to burn deposits if quorum not met
- `burnProposalDepositPrevote` (bool): Whether to burn deposits for prevote failures
- `burnVoteVeto` (bool): Whether to burn deposits for vetoed proposals
- `minDepositRatio` (string): Minimum deposit ratio

```solidity
// SPDX-License-Identifier: LGPL-3.0-only
pragma solidity >=0.8.17;

import "../common/Types.sol";

/// @dev The IGov contract's address.
address constant GOV_PRECOMPILE_ADDRESS = 0x0000000000000000000000000000000000000805;

/// @dev The IGov contract's instance.
IGov constant GOV_CONTRACT = IGov(GOV_PRECOMPILE_ADDRESS);

/**
 * @dev VoteOption enumerates the valid vote options for a given governance proposal.
 */
enum VoteOption {
    // Unspecified defines a no-op vote option.
    Unspecified,
    // Yes defines a yes vote option.
    Yes,
    // Abstain defines an abstain vote option.
    Abstain,
    // No defines a no vote option.
    No,
    // NoWithWeto defines a no with veto vote option.
    NoWithWeto
}

/// @dev WeightedVote represents a vote on a governance proposal
struct WeightedVote {
    uint64 proposalId;
    address voter;
    WeightedVoteOption[] options;
    string metadata;
}

/// @dev WeightedVoteOption represents a weighted vote option
struct WeightedVoteOption {
    VoteOption option;
    string weight;
}

/// @dev DepositData represents information about a deposit on a proposal
struct DepositData {
    uint64 proposalId;
    address depositor;
    Coin[] amount;
}

/// @dev TallyResultData represents the tally result of a proposal
struct TallyResultData {
    string yes;
    string abstain;
    string no;
    string noWithVeto;
}

/// @dev ProposalData represents a governance proposal
struct ProposalData {
    uint64 id;
    string[] messages;
    uint32 status;
    TallyResultData finalTallyResult;
    uint64 submitTime;
    uint64 depositEndTime;
    Coin[] totalDeposit;
    uint64 votingStartTime;
    uint64 votingEndTime;
    string metadata;
    string title;
    string summary;
    address proposer;
}

/// @dev Params defines the governance parameters
struct Params {
    int64 votingPeriod;
    Coin[] minDeposit;
    int64 maxDepositPeriod;
    string quorum;
    string threshold;
    string vetoThreshold;
    string minInitialDepositRatio;
    string proposalCancelRatio;
    string proposalCancelDest;
    int64 expeditedVotingPeriod;
    string expeditedThreshold;
    Coin[] expeditedMinDeposit;
    bool burnVoteQuorum;
    bool burnProposalDepositPrevote;
    bool burnVoteVeto;
    string minDepositRatio;
}

/// @author The Evmos Core Team
/// @title Gov Precompile Contract
/// @dev The interface through which solidity contracts will interact with Gov
interface IGov {
    /// @dev SubmitProposal defines an Event emitted when a proposal is submitted.
    /// @param proposer the address of the proposer
    /// @param proposalId the proposal of id
    event SubmitProposal(address indexed proposer, uint64 proposalId);

    /// @dev CancelProposal defines an Event emitted when a proposal is canceled.
    /// @param proposer the address of the proposer
    /// @param proposalId the proposal of id
    event CancelProposal(address indexed proposer, uint64 proposalId);

    /// @dev Deposit defines an Event emitted when a deposit is made.
    /// @param depositor the address of the depositor
    /// @param proposalId the proposal of id
    /// @param amount the amount of the deposit
    event Deposit(address indexed depositor, uint64 proposalId, Coin[] amount);

    /// @dev Vote defines an Event emitted when a proposal voted.
    /// @param voter the address of the voter
    /// @param proposalId the proposal of id
    /// @param option the option for voter
    event Vote(address indexed voter, uint64 proposalId, uint8 option);

    /// @dev VoteWeighted defines an Event emitted when a proposal voted.
    /// @param voter the address of the voter
    /// @param proposalId the proposal of id
    /// @param options the options for voter
    event VoteWeighted(
        address indexed voter,
        uint64 proposalId,
        WeightedVoteOption[] options
    );

    /// TRANSACTIONS

    /// @notice submitProposal creates a new proposal from a protoJSON document.
    /// @dev submitProposal defines a method to submit a proposal.
    /// @param jsonProposal The JSON proposal
    /// @param deposit The deposit for the proposal
    /// @return proposalId The proposal id
    function submitProposal(
        address proposer,
        bytes calldata jsonProposal,
        Coin[] calldata deposit
    ) external returns (uint64 proposalId);

    /// @dev cancelProposal defines a method to cancel a proposal.
    /// @param proposalId The proposal id
    /// @return success Whether the transaction was successful or not
    function cancelProposal(
        address proposer,
        uint64 proposalId
    ) external returns (bool success);

    /// @dev deposit defines a method to add a deposit to a proposal.
    /// @param proposalId The proposal id
    /// @param amount The amount to deposit
    function deposit(
        address depositor,
        uint64 proposalId,
        Coin[] calldata amount
    ) external returns (bool success);

    /// @dev vote defines a method to add a vote on a specific proposal.
    /// @param voter The address of the voter
    /// @param proposalId the proposal of id
    /// @param option the option for voter
    /// @param metadata the metadata for voter send
    /// @return success Whether the transaction was successful or not
    function vote(
        address voter,
        uint64 proposalId,
        VoteOption option,
        string memory metadata
    ) external returns (bool success);

    /// @dev voteWeighted defines a method to add a vote on a specific proposal.
    /// @param voter The address of the voter
    /// @param proposalId The proposal id
    /// @param options The options for voter
    /// @param metadata The metadata for voter send
    /// @return success Whether the transaction was successful or not
    function voteWeighted(
        address voter,
        uint64 proposalId,
        WeightedVoteOption[] calldata options,
        string memory metadata
    ) external returns (bool success);

    /// QUERIES

    /// @dev getVote returns the vote of a single voter for a
    /// given proposalId.
    /// @param proposalId The proposal id
    /// @param voter The voter on the proposal
    /// @return vote Voter's vote for the proposal
    function getVote(
        uint64 proposalId,
        address voter
    ) external view returns (WeightedVote memory vote);

    /// @dev getVotes Returns the votes for a specific proposal.
    /// @param proposalId The proposal id
    /// @param pagination The pagination options
    /// @return votes The votes for the proposal
    /// @return pageResponse The pagination information
    function getVotes(
        uint64 proposalId,
        PageRequest calldata pagination
    )
        external
        view
        returns (WeightedVote[] memory votes, PageResponse memory pageResponse);

    /// @dev getDeposit returns the deposit of a single depositor for a given proposalId.
    /// @param proposalId The proposal id
    /// @param depositor The address of the depositor
    /// @return deposit The deposit information
    function getDeposit(
        uint64 proposalId,
        address depositor
    ) external view returns (DepositData memory deposit);

    /// @dev getDeposits returns all deposits for a specific proposal.
    /// @param proposalId The proposal id
    /// @param pagination The pagination options
    /// @return deposits The deposits for the proposal
    /// @return pageResponse The pagination information
    function getDeposits(
        uint64 proposalId,
        PageRequest calldata pagination
    )
        external
        view
        returns (
            DepositData[] memory deposits,
            PageResponse memory pageResponse
        );

    /// @dev getTallyResult returns the tally result of a proposal.
    /// @param proposalId The proposal id
    /// @return tallyResult The tally result of the proposal
    function getTallyResult(
        uint64 proposalId
    ) external view returns (TallyResultData memory tallyResult);

    /// @dev getProposal returns the proposal details based on proposal id.
    /// @param proposalId The proposal id
    /// @return proposal The proposal data
    function getProposal(
        uint64 proposalId
    ) external view returns (ProposalData memory proposal);

    /// @dev getProposals returns proposals with matching status.
    /// @param proposalStatus The proposal status to filter by
    /// @param voter The voter address to filter by, if any
    /// @param depositor The depositor address to filter by, if any
    /// @param pagination The pagination config
    /// @return proposals The proposals matching the filter criteria
    /// @return pageResponse The pagination information
    function getProposals(
        uint32 proposalStatus,
        address voter,
        address depositor,
        PageRequest calldata pagination
    )
        external
        view
        returns (
            ProposalData[] memory proposals,
            PageResponse memory pageResponse
        );

    /// @dev getParams returns the current governance parameters.
    /// @return params The governance parameters
    function getParams() external view returns (Params memory params);

    /// @dev getConstitution returns the current constitution.
    /// @return constitution The current constitution
    function getConstitution() external view returns (string memory constitution);
}
```
