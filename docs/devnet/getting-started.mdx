## Default Network Connection Info

| Endpoint / Detail | Example (local node) | Purpose |
|-------------------|----------------------|---------|
| **EVM JSON-RPC** | `http://localhost:8545` | Ethereum-compatible tools (MetaMask, Hardhat, ethers.js) |
| **EVM RPC WS**   | `ws://localhost:8546`   | Streaming real-time, selective data for applications |
| **Cosmos RPC** | `http://localhost:26657` | Tendermint-level interactions (blocks, consensus state) |
| **Cosmos REST API** | `http://localhost:1317` | Query Cosmos SDK modules (bank, staking, etc.) |
| **gRPC** | `localhost:9090` | High-performance Protobuf endpoint for back-ends |
| **Cosmos Chain ID** | `cosmos_9000-1` | Identifier when signing Cosmos transactions |
| **EVM Chain ID** | `9000` | Numeric network identifier for EVM tools |
| **Native Token** | `atom` | Staking / gas token |
| **Denom Exponent** | `6` | Native token has 6 decimal places (`uatom`) |

## Required dependencies (Ubuntu 22.04 LTS)

```bash
sudo apt update && sudo apt install -y \
  build-essential git curl wget jq lz4 liblz4-tool \
  protobuf-compiler pkg-config unzip
````

> *`build-essential` installs **make**, **gcc**, and **g++**, which the Cosmos SDK build scripts require.*

### Go toolchain (1.22 +)

```bash
GO_VERSION=1.22.4
curl -LO "https://go.dev/dl/go${GO_VERSION}.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go${GO_VERSION}.linux-amd64.tar.gz"
echo 'export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin' >> ~/.bashrc
source ~/.bashrc
go version   # verify
```

## Required dependencies (macOS 13/14 via Homebrew)

> **Running a full Cosmos SDK node on iOS is not supported.**
> If you meant an Apple Silicon Mac (Darwin/arm64), install the same tool-chain via Homebrew as shown below.

```bash
# Xcode CLT (headers, make, etc.)
xcode-select --install        # one-time

# Core build & utilities
brew install go git protobuf jq lz4 gnu-tar gnu-sed coreutils
```

* **Go 1.22 +** is installed by Homebrew (`brew install go`).
* `gnu-tar`/`gnu-sed` ensure GNU-compatible flags expected by some Makefiles.

After installing, clone Gaia and build:

```bash
git clone https://github.com/cosmos/gaia.git
cd gaia
make install    # drops 'gaiad' into $(go env GOPATH)/bin
```

You now have all packages needed to compile and run **gaiad** (or any other Cosmos-SDK-based chain) on Ubuntu or macOS.

## Using Remote Endpoints

The `evmd` CLI and other clients can target remote public nodes.

### Method 1 — configure **evmd CLI**

Edit `~/.evmd/config/client.toml`:

```toml
# ~/.evmd/config/client.toml
node = "tcp://rpc.example.com:443"
```

All subsequent `evmd` commands use the remote node:

```sh
evmd query bank balances <cosmos_bech32_address>
```

### Method 2 — direct API calls

#### EVM JSON-RPC

```js
import { ethers } from "ethers";

const provider = new ethers.JsonRpcProvider("https://rpc.example-evm.com");
const balance = await provider.getBalance("0x...");
```

#### Cosmos REST API

```sh
curl https://api.example-cosmos.com/cosmos/staking/v1beta1/params
```
