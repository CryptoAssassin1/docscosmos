## Required dependencies (Ubuntu 22.04 LTS)

```bash
sudo apt update && sudo apt install -y \
  build-essential git curl wget jq lz4 liblz4-tool \
  protobuf-compiler pkg-config unzip
````

> *`build-essential` installs **make**, **gcc**, and **g++**, which the Cosmos SDK build scripts require.*

### Go toolchain (1.22 +)

```bash
GO_VERSION=1.22.4
curl -LO "https://go.dev/dl/go${GO_VERSION}.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go${GO_VERSION}.linux-amd64.tar.gz"
echo 'export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin' >> ~/.bashrc
source ~/.bashrc
go version   # verify
```

## Required dependencies (macOS 13/14 via Homebrew)

> **Running a full Cosmos SDK node on iOS is not supported.**
> If you meant an Apple Silicon Mac (Darwin/arm64), install the same tool-chain via Homebrew as shown below.

```bash
# Xcode CLT (headers, make, etc.)
xcode-select --install        # one-time

# Core build & utilities
brew install go git protobuf jq lz4 gnu-tar gnu-sed coreutils
```

* **Go 1.22 +** is installed by Homebrew (`brew install go`).
* `gnu-tar`/`gnu-sed` ensure GNU-compatible flags expected by some Makefiles.

After installing, clone Gaia and build:

```bash
git clone https://github.com/cosmos/gaia.git
cd gaia
make install    # drops 'gaiad' into $(go env GOPATH)/bin
```

You now have all packages needed to compile and run **gaiad** (or any other Cosmos-SDK-based chain) on Ubuntu or macOS.

### Build the binary

```sh
git clone https://github.com/cosmos/evm.git
cd evm
make build
````

### Start the local testnet

```sh
make start-localnet-ci
```

This launches a single-node network with pre-funded accounts and exposes all local endpoints.
Connect EVM wallets and other tools to `http://localhost:8545`.
