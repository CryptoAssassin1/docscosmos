## Icon Wrapper

The `IconWrapper` component provides a consistent background and hover effects for icons.

<div className="flex items-center gap-4 p-6 bg-slate-50 dark:bg-slate-900 rounded-lg mb-6">
  <IconWrapper color="#4B47CA" size="sm">
    <CosmosIcon size={16} color="white" />
  </IconWrapper>
  
  <IconWrapper color="#627EEA" size="md" gradient>
    <EthereumIcon size={20} color="white" />
  </IconWrapper>
  
  <IconWrapper color="#22E2A8" size="lg">
    <IBCIcon size={24} color="white" />
  </IconWrapper>
  
  <IconWrapper color="#7552FF" size="xl" gradient>
    <SDKIcon size={28} color="white" />
  </IconWrapper>
</div>

```jsx
import { IconWrapper, CosmosIcon } from '/snippets/icons.mdx'

<IconWrapper color="#4B47CA" size="md" gradient>
  <CosmosIcon size={20} color="white" />
</IconWrapper>
```

## Props Reference

### Icon Components

All icon components accept these props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `24` | Size in pixels |
| `color` | `string` | `"currentColor"` | Icon color (hex, rgb, or CSS color) |
| `className` | `string` | `""` | Additional CSS classes |

### IconWrapper Component

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Icon component to wrap |
| `color` | `string` | `"#4B47CA"` | Background color |
| `size` | `"sm" | "md" | "lg" | "xl"` | Wrapper size |
| `gradient` | `boolean` | `false` | Enable gradient background |

## Color Palette

Use these Cosmos brand colors for consistency:

- **Primary**: `#4B47CA` (Cosmos Deep Blue)
- **Light**: `#7552FF` (Cosmos Purple)  
- **Accent**: `#22E2A8` (Cosmos Mint Green)
- **Ethereum**: `#627EEA` (Ethereum Blue)
- **API**: `#FF5628` (Orange Red)
