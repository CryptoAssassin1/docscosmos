## Quick Reference

### Common Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/cosmos/bank/v1beta1/balances/{address}` | GET | Query account balances |
| `/cosmos/staking/v1beta1/validators` | GET | List all validators |
| `/cosmos/gov/v1beta1/proposals` | GET | Query governance proposals |
| `/cosmos/distribution/v1beta1/delegators/{address}/rewards` | GET | Query staking rewards |

### Common CLI Commands

```bash
# Query account balance
cosmos query bank balances <address>

# Query validator set
cosmos query staking validators

# Submit a transaction
cosmos tx bank send <from> <to> <amount> --chain-id <chain-id>

# Query transaction by hash
cosmos query tx <hash>
```
