## Real-Time Data with WebSocket

<CodeGroup>
```javascript Subscribe to New Blocks
const WebSocket = require('ws');
const ws = new WebSocket('wss://evm-rpc.cosmos.network');

ws.on('open', () => {
  ws.send(JSON.stringify({
    jsonrpc: '2.0',
    method: 'eth_subscribe',
    params: ['newHeads'],
    id: 1
  }));
});

ws.on('message', (data) => {
  const response = JSON.parse(data);
  console.log('New block:', response.params.result);
});
```

```javascript Subscribe to Logs
const ws = new WebSocket('wss://evm-rpc.cosmos.network');

      topics: ['0x...']

  }));
});
```

## Performance Considerations

### Public RPC Limits
- **Rate Limiting**: Public endpoints have request limits
- **Payload Size**: Large requests may be rejected
- **Concurrent Connections**: WebSocket connection limits apply

### Optimization Tips
- Use `latest` block tag sparingly in production
- Batch multiple requests when possible
- Implement proper retry logic with exponential backoff
- Consider running your own node for high-volume applications
