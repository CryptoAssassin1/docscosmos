## Overview

Returns the value from a storage position at a given address

### Parameters

1. `address` - Parameter description
2. `position` - Parameter description
3. `block_number` - Parameter description

### Returns

Storage value

## Examples

### Request

<CodeGroup>
```bash cURL
curl -X POST https://evm-rpc.cosmos.network \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getStorageAt",
    "params": ["0x742d35cc6644c068532fddb11B4C36A58D6D3eAb", "0x0", "latest"],
    "id": 1
  }'
```

```javascript JavaScript (Web3.js)
const Web3 = require('web3');
const web3 = new Web3('https://evm-rpc.cosmos.network');

// Example call
const result = await web3.eth.getStorageAt(0x742d35cc6644c068532fddb11B4C36A58D6D3eAb, 0x0, latest);
console.log(result);
```

```javascript JavaScript (Ethers.js)
const { ethers } = require('ethers');
const provider = new ethers.providers.JsonRpcProvider('https://evm-rpc.cosmos.network');

// Example call
const result = await provider.send('eth_getStorageAt', ["0x742d35cc6644c068532fddb11B4C36A58D6D3eAb", "0x0", "latest"]);
console.log(result);
```

```python Python (Web3.py)
from web3 import Web3

w3 = Web3(Web3.HTTPProvider('https://evm-rpc.cosmos.network'))

// Example call
result = w3.manager.request_blocking('eth_getStorageAt', ['0x742d35cc6644c068532fddb11B4C36A58D6D3eAb', '0x0', 'latest'])
print(result)
```
</CodeGroup>
