## Overview

Executes a new message call immediately without creating a transaction

### Parameters

1. `transaction` - Parameter description
2. `block_number` - Parameter description

### Returns

Call result

## Examples

### Request

<CodeGroup>
```bash cURL
curl -X POST https://evm-rpc.cosmos.network \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_call",
    "params": [{"to": "0x742d35cc6644c068532fddb11B4C36A58D6D3eAb", "value": "0x1000000000000000000", "gas": "0x5208"}, "latest"],
    "id": 1
  }'
```

```javascript JavaScript (Web3.js)
const Web3 = require('web3');
const web3 = new Web3('https://evm-rpc.cosmos.network');

// Example call
const result = await web3.eth.call({'to': '0x742d35cc6644c068532fddb11B4C36A58D6D3eAb', 'value': '0x1000000000000000000', 'gas': '0x5208'}, latest);
console.log(result);
```

```javascript JavaScript (Ethers.js)
const { ethers } = require('ethers');
const provider = new ethers.providers.JsonRpcProvider('https://evm-rpc.cosmos.network');

// Example call
const result = await provider.send('eth_call', [{"to": "0x742d35cc6644c068532fddb11B4C36A58D6D3eAb", "value": "0x1000000000000000000", "gas": "0x5208"}, "latest"]);
console.log(result);
```

```python Python (Web3.py)
from web3 import Web3

w3 = Web3(Web3.HTTPProvider('https://evm-rpc.cosmos.network'))

// Example call
result = w3.manager.request_blocking('eth_call', [{'to': '0x742d35cc6644c068532fddb11B4C36A58D6D3eAb', 'value': '0x1000000000000000000', 'gas': '0x5208'}, 'latest'])
print(result)
```
</CodeGroup>
