# EVM JSON-RPC API

The EVM module provides full Ethereum JSON-RPC compatibility, allowing you to use existing Ethereum tooling with Cosmos EVM chains.

## Endpoint

```
https://evm-rpc.cosmos.network
```

## Standard Methods

### eth_getBalance

Get the balance of an account.

**Parameters:**
1. `address` - 20-byte address
2. `block` - Block number or "latest", "earliest", "pending"

```bash
curl -X POST \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_getBalance",
    "params": ["0x742d35Cc6635C0532925a3b8D20d8B6C2B0A3d", "latest"],
    "id": 1
  }' \
  https://evm-rpc.cosmos.network
```

**Response:**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1bc16d674ec80000"
}
```

### eth_sendTransaction

Send a transaction.

**Parameters:**
1. `transaction` - Transaction object

```bash
curl -X POST \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_sendTransaction",
    "params": [{
      "from": "0x742d35Cc6635C0532925a3b8D20d8B6C2B0A3d",
      "to": "0x162d35Cc6635C0532925a3b8D20d8B6C2B0A3d", 
      "value": "0x9184e72a000",
      "gas": "0x5208",
      "gasPrice": "0x9502f9000"
    }],
    "id": 1
  }' \
  https://evm-rpc.cosmos.network
```

### eth_call

Execute a message call.

**Parameters:**
1. `transaction` - Transaction call object
2. `block` - Block number or "latest", "earliest", "pending"

```bash
curl -X POST \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_call",
    "params": [{
      "to": "0x162d35Cc6635C0532925a3b8D20d8B6C2B0A3d",
      "data": "0x70a08231000000000000000000000000742d35Cc6635C0532925a3b8D20d8B6C2B0A3d"
    }, "latest"],
    "id": 1
  }' \
  https://evm-rpc.cosmos.network
```

## Block Methods

### eth_blockNumber

Get the latest block number.

```bash
curl -X POST \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }' \
  https://evm-rpc.cosmos.network
```

### eth_getBlockByNumber

Get block information by number.

```bash
curl -X POST \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_getBlockByNumber",
    "params": ["latest", true],
    "id": 1
  }' \
  https://evm-rpc.cosmos.network
```

## Error Codes

| Code | Message | Description |
|------|---------|-------------|
| -32700 | Parse error | Invalid JSON |
| -32600 | Invalid Request | JSON is not a valid request object |
| -32601 | Method not found | Method does not exist |
| -32602 | Invalid params | Invalid method parameters |
| -32603 | Internal error | Internal JSON-RPC error |

## Rate Limiting

The JSON-RPC API is rate limited to:
- 1000 requests per minute for read operations
- 100 requests per minute for write operations
