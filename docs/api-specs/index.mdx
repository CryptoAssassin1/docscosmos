## Available Specifications

### Cosmos EVM

The Cosmos EVM API specification includes:

- **JSON-RPC Methods**: Ethereum-compatible endpoints (`eth_*`, `net_*`, `web3_*`)
- **REST Endpoints**: Cosmos SDK query and transaction endpoints
- **WebSocket Subscriptions**: Real-time event streaming
- **Schema Definitions**: Request/response models and data types

### Using the Specifications

These OpenAPI specifications can be used with various tools:

- **Code Generation**: Generate client SDKs in multiple languages
- **API Testing**: Import into Postman, Insomnia, or similar tools
- **Documentation**: Generate interactive API documentation
- **Validation**: Validate requests and responses against schemas

```bash
# Example: Generate TypeScript client
npx @openapitools/openapi-generator-cli generate \
  -i docs/api-specs/cosmos-evm.json \
  -g typescript-axios \
  -o ./generated-client
```
